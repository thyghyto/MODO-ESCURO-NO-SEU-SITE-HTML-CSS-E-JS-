# 🌙 Modo Escuro

Um exemplo simples de como criar um botão para ativar e desativar o modo escuro em uma página.

---

## 🟢 Versão simples

A versão básica funciona apenas com um botão que liga e desliga o modo escuro da página.
Ela é fácil de entender e usa poucas linhas de código.

---

## 🎨 Versão mais bonita

A versão mais estilizada melhora o visual do botão.
Ela deixa o botão redondo, com sombra e animações ao passar o mouse, além de mudar a aparência no modo escuro.

---

## 🧠 Como funciona

O botão ativa ou desativa o modo escuro mudando as cores da página automaticamente.

---

## 🚀 Tecnologias usadas

* HTML
* CSS
* JavaScript
